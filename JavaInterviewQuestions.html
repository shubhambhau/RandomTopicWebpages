What is Custom Exception in java

    • A custom exception is a user defined class that extends Exception or Runtime Exception to represent application specific error conditions.
      It helps make error handling more meaningful and readable by giving descriptive names to exceptions.
      We create it by defining a class and throwing it where that specific error scenario occurs.

    • What is the difference between the Compile time exception and Runtime Exception

    • A Compile time exception also known as a checked exception, is checked by the compiler and must be either handled with try-catch or declared with throws.
      These are often explicitly thrown by the programmer, such as IOException or SQLException.
      A Runtime Exception, also known as an unchecked exception, does not need to be handled or declared, for example ArithmeticException or NullPointerException./ol>

        2. What is the difference between ArrayList and HashSet

        2. ArrayList stores elements in an ordered sequence, allows duplicates, and is index-based, so you can access elements by position.
          HashSet stores unique elements only, does not maintain insertion order, and is based on hashing for faster lookups.
          Use ArrayList when order and duplicates matter, and HashSet when uniqueness and fast search are priorities.
